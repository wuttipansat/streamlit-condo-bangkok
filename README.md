# Condo Rental Prediction Along BTS Silom
 
A streamlit web application for estimating monthly condo rental prices along the BTS Silom area in Bangkok.
The app uses a traned machine learning model to predict rental prices from condo dataset including, unit type, room size, etc.

![Image alt](https://github.com/wuttipansat/streamlit-condo-bangkok/blob/08f217278bc6d2f3ee83cfcf099879d828354848/snapshot1.png)

## Project Overview

This project builds and deploys a rental price prediction model usiing Bangkok condo along BTS Silom dataset.
Users can input condo details through a Streamlit interface and receive an estimated monthly rental price in THB/month.

## Features

- Interactive Streamlit web app
- Condo rental price prediction
- Trained machine learning model loaded from saved artifacts
- Model training pipeline included

## Getting Started

Run the app locally with:

```bash
streamlit run app.py
```
